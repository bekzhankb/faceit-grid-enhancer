# FACEIT Grid Enhancer

![Status](https://img.shields.io/badge/status-development-yellow)
![Version](https://img.shields.io/badge/version-1.0.0-blue)

**FACEIT Grid Enhancer** is a powerful browser extension designed to enhance the user experience on the FACEIT platform. This project is currently in its **early development stage**, with core modules being actively implemented and refined.

*We(me and my delulu) are building a tool that provides tactical analytics, automation for routine tasks, and quality-of-life (QoL) features directly within the match room.*

---
##  Key Features

* **AI Deep Map Veto**: Smart recommendations for map picks and bans based on tactical analysis.
* **Smurf Detector**: Identify suspicious accounts to help evaluate match fairness.
* **Personal Dodge List**: Maintain a local blacklist of players with visual indicators in the lobby.
* **General QoL**: Streamlined interface improvements and quick access to player/hub information.

##  Tech Stack

* **Manifest V3**: Using the latest Chrome extension architecture.
* **JavaScript (ES6+)**: Asynchronous logic and efficient DOM manipulation.
* **Chrome Storage API**: Secure local storage for user settings and data.
* **CSS3**: Custom styles perfectly integrated with the FACEIT design language.

##  Installation

1.  Clone this repository or download it as a ZIP:
    `git clone https://github.com/bekzhankb/faceit-grid-enhancer.git`
2.  Open your browser (Chrome/Edge/Brave) and go to: `chrome://extensions/`.
3.  Enable **"Developer mode"** in the top right corner.
4.  Click **"Load unpacked"** and select the project folder.

## Configuration
Manage all features via the pop-up menu. You can toggle specific modules on or off in real-time without needing to reload.
